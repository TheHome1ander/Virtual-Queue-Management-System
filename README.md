This is a Queue Management App.
Shop Owners can register their shops in the app. Each owner can register multiple shops. The request has to get approved by the admin.
The owner would then get access to a dashboard for each of his shops from where he can take actions like Call next, Reset Queue, No show, Open/Close shop.
The owner shares his shops link with his customers through the dashboard. Using this link, the customers can join the queue. Queue numbers update real-time. The customer gets
a time window to show up. If they fail to do so, they get pushed back to the end of the queue. The time window can be set by the person managing the dashboard.


This project was made with the hope of minimizing a real-world problem - long wait times in queues.
